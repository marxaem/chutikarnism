Spotify Playlists Recommendation

Created a simple playlist recommendation system that relied only on tracks in each playlist.

November 2021
  • The dataset is a part of Spotify Million Playlist Dataset Challenge. The dataset contains playlist information such as playlist name, track name, artist, etc.
  • To create a recommendation system, started by extracting keywords from playlist information which includes artists, track names, and album names. Then converted text into vector forms using CountVectorizer and, lastly, calculated their similarities.
